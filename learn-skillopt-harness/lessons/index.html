SkillOpt & Agent Harnesses

Train reusable skills for your frozen coding agent — pi, Claude Code, Codex, or Kiro — on real software development tasks, without touching model weights.

Hands-On: SkillOpt-Sleep

  • 03 Sleep Cycle on Your Real Sessions skillopt-sleep dry-run against your Claude Code / Codex transcripts; review a staged proposal. ~20 min
Hands-On: The Full Training Loop
  • 04 A Full Training Run Shipped benchmark + claude_code_exec backend, end to end, producing a best_skill.md. ~25 min
  • 05 A Custom Env for Your Real Task dataloader.py + rollout.py + initial.md for one of your own software-dev workflows. ~30 min
Capstone
  • 06 Wire In Your Own Harness (pi or Kiro) Implement the backend contract using codex_exec as a template; or a Kiro MCP shell for Sleep. ~40 min
