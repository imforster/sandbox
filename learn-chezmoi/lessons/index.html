chezmoi

Managing three macOS machines from one dotfiles repo — with machine-specific brew packages, credentials, and shell configs.

Getting Started
  • 01 Init & Add Install chezmoi, initialize from your existing dotfiles repo, and bring your first file under management. ~10 min
  • 02 Templates & Machine Detection Use Go templates and .chezmoi variables to render different configs per machine. ~12 min
  • 03 Custom Data Files Define machine_type, package lists, and feature flags in .chezmoidata.toml. ~10 min
Automation
  • 04 Scripts & Brew Bundles Auto-install the right brew packages per machine with templated run_onchange scripts. ~12 min
  • 05 Secrets & macOS Keychain Pull credentials from Keychain into templates — nothing sensitive in git. ~10 min
  • 06 Ignore & Exclude Conditionally skip files per machine using .chezmoiignore as a template. ~8 min
Daily Use
  • 07 Edit Workflow & Daily Operations The edit → diff → apply → commit cycle and pulling changes across machines. ~8 min
  • 08 Run Scripts Deep Dive All script types: run_once, run_onchange, run_before, run_after — and idempotency patterns. ~10 min
Advanced
  • 09 External Files & Archives Pull shell plugins, fonts, and binaries from URLs with .chezmoiexternal.toml. ~10 min
  • 10 Bootstrap a New Machine One command to set up a fresh Mac — everything from lessons 1-9 fires automatically. ~12 min
