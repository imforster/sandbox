Agentic Skills Best Practices

A structured course on designing, building, and governing agentic skills files — from architecture to anti-patterns, grounded in the ten-pillar framework and AWS enterprise guidance.

Foundations
  • 01 Architecture and Structure Organize skill files into metadata, interfaces, logic, workflows, and configuration sections. ~10 min
  • 02 Documentation Clarity Write self-documenting skills with triggers, examples, limitations, and dependencies. ~8 min
  • 03 Scope Definition Apply Single Responsibility to skills — one domain, one task family, explicit boundaries. ~8 min
  • 04 Modularity and Reusability Design composable skills with shared libraries, dependency injection, and include directives. ~10 min
Craft
  • 05 Prompt Engineering Within Skills Chain of Thought, ReAct, self-reflection — advanced prompting patterns for skill files. ~12 min
  • 06 Tool Integration Patterns MCP protocol, tool gateway architectures, retry logic, and enterprise access control. ~12 min
  • 07 Testing and Validation Unit tests, integration tests, evaluation frameworks, and quality gates for skills. ~10 min
Operations & Scale
  • 08 Version Control and Maintenance Semantic versioning, changelogs, dependency management, and deprecation timelines. ~8 min
  • 09 Performance Optimization Token reduction, prompt caching, context compression, and performance metrics. ~10 min
  • 10 Anti-Patterns and Pitfalls Seven common mistakes — monolithic skills, hard-coded config, missing error handling, and more. ~10 min

Reference: Agentic Skills Glossary

Total course time: ~98 minutes · 10 lessons · Interactive quizzes in each
