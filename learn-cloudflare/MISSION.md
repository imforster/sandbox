# Mission

## Why am I learning this?

I already know Cloudflare can host static websites. I want to explore what else is possible — specifically what I can build and run on Cloudflare's **free tier** while getting the security and speed benefits that Cloudflare is known for.

## What does success look like?

I can confidently answer: "What can I host on Cloudflare for free beyond a static site?" — and I've built at least one working example of something dynamic running on the free tier, protected by Cloudflare's security layer and served at edge speed. I can also set up Google OAuth login and point my Google Workspace subdomain to a Worker with full SSL and DDoS protection.

## Current state

- I understand static site hosting on Cloudflare
- I want the security benefits (DDoS protection, WAF, SSL) and speed (edge network, caching)
- I'm curious about the boundary between "static site" and "dynamic application" on this platform
- I want to stay within free-tier limits (cost = $0)

## Constraints

- Free tier only — no paid plans
- Exploration-driven — no specific project deadline
- Breadth first, then depth — understand the landscape of what's possible before going deep on any one thing

## Evolution

- *2026-07-02*: Mission established. Starting from "what's beyond static?" with security and speed as key motivators.
